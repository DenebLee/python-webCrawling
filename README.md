# Python 이미지 크롤링


## 실행방법👀👀👀👀
1. 파이썬 설치 https://www.python.org/downloads/ (파이썬 설치시 path에 추가 라디오 버튼 꼭 클릭) 
2. vscode python extension pack 설치(필요하면)
3. 필요한 라이브러리 설치 <br>
  3.1 pip install selenium <br>
  3.2 pip install pygame <br>
  3.3 pip install image <br>
4. f5로 컴파일및 실행 
<br>


##  개발환경🌹

- Visual Studio Code
- Python 3.9.4 
- Selenium 3.141.0
